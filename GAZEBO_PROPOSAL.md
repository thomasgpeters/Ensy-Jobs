# Gazebo Installation Proposal

**Client:** Elsy Sandora
**Date:** April 16, 2026

---

## Project Overview

Install a client-purchased 12x10 gazebo kit on the existing 12x12 brick paver patio. The gazebo will be anchored using four short 4x4 pressure-treated posts set in concrete. The gazebo legs are designed to fit over the 4x4 posts. The gazebo does not need to be centered on the patio.

---

## Scope of Work

### 1. Positioning the Gazebo

- Determine final placement of the 12x10 gazebo on the 12x12 patio
- Mark the four anchor post locations to match the gazebo leg positions
- Verify layout is square and properly aligned
- Identify any patio blocks that need to be removed for post placement (up to 4 pavers may need to come up for one post location)

### 2. Digging Post Holes

- Remove patio pavers as needed at each post location
- Excavate four post holes beneath the paver surface to proper depth for anchoring
- Clear any gravel base material or sub-base as needed for clean holes

### 3. Setting and Leveling Posts with Concrete

- Set four short 4x4 pressure-treated posts into the holes
- Pour concrete around each post
- Level and square all four posts to ensure the gazebo sits true and plumb
- Allow concrete to cure before proceeding with assembly
- Reinstall or patch any displaced pavers around the posts

### 4. Gazebo Assembly

- Unpack and inventory all kit components
- Assemble the 12x10 gazebo per manufacturer instructions
- Fit gazebo legs over the four 4x4 anchor posts for a secure connection
- Verify structure is level, square, and stable
- Clean up all packaging and job site debris
