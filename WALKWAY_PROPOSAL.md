# Walkway with Stone Blocks Proposal

**Client:** Elsy Sandora
**Date:** April 16, 2026

---

## Project Overview

Replace the existing single-wide stepping stone walkway with a double-wide 15" block walkway. Blocks will be laid tightly together with minimal gaps from the bottom of the walk all the way up to the front door. The three existing makeshift steps in the middle of the slope will be rebuilt properly in the same stacked-block style, and a fourth step with a longer landing will be added above them. All steps will have proper footing to remain stable.

---

## Existing Conditions

- Long curving walkway running from the street/sidewalk up to the covered front porch
- Currently single-wide (~15" square pavers) laid single file with large gaps between blocks
- The yard slopes upward toward the house — grade change is handled by three makeshift steps built from stacked stone blocks in the middle section
- Solar path lights installed along the garden bed side
- Established plantings on both sides: hostas, phlox, ground cover, and other perennials
- Mulch bed on the garden side, lawn on the opposite side

---

## Scope of Work

### 1. Removal of Existing Walkway

- Carefully remove all existing single-wide pavers along the full walkway length
- Set aside any reusable blocks
- Note: Elsy will remove all plantings lining both sides of the walkway prior to work

### 2. Base Preparation

- Excavate and widen the walkway bed to accommodate double-wide blocks (approximately 30" wide)
- Grade the base to follow the natural curve and slope of the walkway
- Compact sub-base and add gravel/crushed stone layer for a stable, level foundation
- Ensure proper drainage so water does not pool on the walkway

### 3. Double-Wide Block Walkway Installation

- Lay two 15" blocks side by side along the full length of the walkway
- Fit all blocks tightly together with minimal gaps from bottom to front door
- Follow the existing curve of the walkway path
- Fill joints with polymeric sand or appropriate joint material for a tight, finished look

### 4. Step Section — Rebuild and Expand

The existing three makeshift steps handle the grade change in the middle of the walkway. Elsy wants to keep this stacked-block concept but done right, and add one more step.

**Footing:**
- Excavate beneath the step section to install proper footing
- Pour concrete footing or lay a compacted gravel foundation to keep all four steps stable on the slope
- Footing must support the full width of the double-wide walkway

**Steps (4 total):**
- Rebuild the three existing steps using stacked 15" blocks, properly leveled and aligned
- Add a fourth step above the existing three
- The fourth (top) step will have a longer landing — providing a wider, more stable transition before the walkway continues to the front door
- All steps double-wide to match the walkway width
- Each step rise should be consistent for safe, even footing

### 5. Finishing

- Reinstall solar path lights along the walkway
- Fill all joints and step edges with polymeric sand
- Clean up site debris
- Final inspection for level, alignment, and stability
